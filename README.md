# Cyberpunk Todo App

A simple todo application with a cyberpunk theme.

## Description

This project is a basic todo app built using HTML, CSS, and JavaScript. It allows users to add, delete, and mark tasks as complete. The app features a cyberpunk-inspired design with a dark color scheme and neon green accents.

## Technologies Used

*   HTML
*   CSS
*   JavaScript

## Folder Structure

```
cyberpunk-todo-app/
├── index.html
├── style.css
├── script.js
├── vercel.json
└── README.md
```

*   `index.html`: Main HTML file for the todo app, containing the structure and content.
*   `style.css`: CSS file for styling the todo app with a cyberpunk theme.
*   `script.js`: JavaScript file for handling todo app logic (adding, deleting, marking as complete).
*   `vercel.json`: Configuration file for Vercel deployment, specifying build and routing settings.
*   `README.md`: Project documentation and instructions.

## Usage

1.  Clone the repository:

    ```bash
    git clone [repository URL]
    ```

2.  Open `index.html` in your browser.

## Vercel Deployment

This project is configured for deployment on Vercel. The `vercel.json` file contains the necessary configuration for building and routing.

To deploy to Vercel:

1.  Install the Vercel CLI:

    ```bash
    npm install -g vercel
    ```

2.  Deploy from your project directory:

    ```bash
    vercel
    ```
